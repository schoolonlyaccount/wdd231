My Website Plan

Site Name

Kell's Video Game Club - This name reflects a community-focused site where users can share and discover video game recommendations.

Site Purpose

The site provides a way to view recommended games from others, along with the ability to submit your own recommended games.

Scenarios

  • How can I submit my own recommended games?
  • How do I see who recommended what?

Color Scheme

Primary (#5043c0) color will be used for the header and footer.

Secondary (#558488) color will be used for the headings.

White (#ffffff) color will be used for the text.

Typography

Roboto will be the primary font used throughout the site.
